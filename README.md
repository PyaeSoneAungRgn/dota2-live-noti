# Dota2 Live Noti

Seamless Access to Upcoming Dota2 Fixtures, Right from Your Desktop Menubar!

## Installation

Visit the [Release page](https://github.com/PyaeSoneAungRgn/dota2-live-noti/releases) and Download for Your Operating System..

| ------------- | ------------- |
| Linux  | ✅ |
| macOS  | ✅ |
| Windows  | ✅ |
